Automatisez la création de vos One-Pagers PowerPoint avec un moteur Rust ultra-rapide et une interface intuitive.

Télécharger l'application Voir le code source

Vitesse Rust

Un moteur natif conçu pour traiter des milliers de lignes de données et générer des dizaines de PPTX en quelques secondes.

Multi-langues

Support natif pour FR, EN, DE, IT, ES. Détection automatique des langues dans vos dossiers de templates.

Smart Mappings

Associez n'importe quelle colonne CSV à vos tags PowerPoint dynamiquement, sans modifier vos fichiers sources.

Guide de démarrage

Pré-requis Importants

  • • Dossiers en local : Les fichiers doivent être téléchargés sur votre machine (pas uniquement sur le Drive/Cloud).
  • • Matériel : Mac Apple Silicon (processeur M1 minimum) recommandé.

Gestion des Templates

L'application scanne intelligemment la structure de votre dossier :

  • • Structure : Un sous-dossier par langue (ex: FR, EN) avec les fichiers .pptx à l'intérieur.
  • • Flexibilité : Désactivez les langues dont vous n'avez pas besoin directement depuis l'interface.

Premier Lancement (macOS)

Si macOS bloque l'ouverture, utilisez cette commande dans le Terminal :

xattr -cr /Applications/OnePagerGenerator.app
